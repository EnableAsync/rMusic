## ✨ 功能
- [x] 基于 [UnblockNeteaseMusic/server-rust](https://github.com/UnblockNeteaseMusic/server-rust) 实现解析音乐下载链接
- [x] 基于 [Tauri](https://tauri.app) 实现音乐在线播放
- [ ] 上传音乐至云盘
